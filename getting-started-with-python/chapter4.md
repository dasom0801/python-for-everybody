# CHAPTER 4: 함수

## Part 1. 함수의 개념

+ 함수는 코드를 저장했다가 재사용하는 것

+ 프로그래밍에서 중요한 한 가지는 같은 작업을 반복하지 않는 것

  ```python
  def thing():
      print('Hello')
      print('Fun')
  thing()
  print('Zip')
  thing()
  ```

+ `def` 

  + 정의된 함수를 의미
  + 저장단계, 코드를 생성하여 기록함

+ 내장함수

  + 파이썬이 설치될 때부터 정의되어있었던 함수
  + print(), input(), type(), float(), int()...

## Part 2. 함수 만들기

+ 함수 호출
  + def는 저장하는 것, 호출은 재사용하는 것
+ 매개 변수
  + 함수 안으로 전달되는 값
  + 함수가 실행되는 동안에만 존재함
  + 매개 변수를 설정한 만큼 인자로 전달하지 않으면 traceback 출력
+ 반환
  + 반환문을 만나면 함수 종류
  + 함수가 끝났을 때 나타날 값
